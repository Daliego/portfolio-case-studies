##### Resumo (Português)

A maioria das competições surge como um desafio ou brincadeira entre um grupo, permitindo a integração entre diferentes áreas do conhecimento e culminando na criação de novas tecnologias. Essa é a origem da competição **Micro Mouse**, que com o tempo ganhou grande visibilidade no mundo. Entretanto, no Brasil, limitações de recursos financeiros — tanto para adquirir o robô rato quanto para construir um labirinto — reduzem o acesso e afastam grande parte das pessoas desse ambiente. Em contrapartida, ferramentas gratuitas surgiram para quebrar essas barreiras físicas e democratizar o acesso a labirintos e a uma versão virtual do robô, além de permitir liberdade criativa em múltiplas linguagens de programação. Com o uso do simulador digital **MMS (Micro Mouse Simulator)**, foi possível desenvolver algoritmos usados no ambiente competitivo, explorar recursos de análise de desempenho do código e testar soluções em diversos labirintos disponíveis no software.

**Palavras-chave:** Robô rato, resolução de labirintos, algoritmos, simulador digital, liberdade de linguagem.

##### Abstract (English)

Most competitions arise as a challenge or as a game between a group, which allows integration between different areas of knowledge, culminating in the creation of a new technology. This is the origin of the **Micro Mouse** competition, which over time has gained increasing visibility worldwide. However, in Brazil, limited financial resources—both to acquire the mouse robot and to build a maze—exclude a large portion of people from this environment. Despite this, free tools have helped break down these physical barriers and democratize access to both mazes and a virtual version of the robot, while enabling creative freedom across multiple programming languages. By using a digital simulator, it was possible to implement algorithms commonly used in the competitive environment, analyze code performance within the software to improve logic, and represent the real robot with high reliability.

**Keywords:** Mouse robot, maze solving, algorithms, digital simulator, freedom of language.
